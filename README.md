# TikTok video downloader

Project URL: https://tiktokgrabber.reasadazim.com

This is a Django app which have following capabilities:
 - [x] Download TikTok video
 - [x] Download TikTok story
## Framework
 - Django Framework 4.2.4
## Dependencies
 - Python 3.11.4
 - PIP 23.2.1
## Tested in
 - MariaDB 10.4.24
 - PhpMyAdmin 5.2.0
